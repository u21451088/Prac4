For the Application to work one should have a running instance of the sakila database running on their machine and a mysql user who can access said database.
The application uses enviromnemt variables to access the database, hence the marker should also have the variables specified in the specifications in their system variables.

To get the application running the following must be done:
1. Download From GitHub at https://github.com/u21451088/Prac4.git or download submitted files
2. Have a verion of netbeans 13+ installed
3. Open Nebeans and click file > open Project and navigate to the downloaded file
4. once open in IDE right click on the project and select the "Build" option
5. Once built one should be able to run the project by either clicking F6 or by manually running the Prac04.java file
